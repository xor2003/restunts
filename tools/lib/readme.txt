the object files here are extracted from cm.lib using tlib.exe.

cm.lib is the medium model libc from borland c++ 5.2.
tlib.exe is a borland utility for manipulating lib files.

example commands:
	tlib cm.lib * GETVECT
	tlib cm.lib * LABS
	tlib cm.lib * STRCPY
	tlib cm.lib * MEMCPY
	tlib cm.lib * FMEMCPY
	tlib cm.lib * H_LDIV
	tlib cm.lib * F_LXMUL
	tlib cm.lib * F_SCOPY
	tlib cm.lib * H_LRSH
	tlib cm.lib * H_PADD
	tlib cm.lib * H_PINA
	tlib cm.lib * H_PADA
	tlib cm.lib * N_PCMP
	tlib cm.lib * F_PCMP
	tlib cm.lib * H_LURSH
	tlib cm.lib * H_PSBP
	tlib cm.lib * H_LLSH
